# 铸绿筑梦党建品牌

> 来源：园林与艺术学院党委 | 主讲：刘彩云 党委副书记

## 品牌定位

江西农业大学"金扁担"党建品牌的学院子品牌。

## 核心内涵：一根扁担挑两头

```
┌─────────────────────────────────────┐
│           铸绿筑梦                   │
├─────────────────┬───────────────────┤
│    扁担一头      │     扁担另一头      │
├─────────────────┼───────────────────┤
│      树木        │       树人         │
│  （森林树木）     │   （立德树人）      │
└─────────────────┴───────────────────┘
```

- **铸绿**：强化生态保护使命
- **筑梦**：落实立德树人根本任务

## 四梁八柱体系

### 四梁（四个维度）
1. 政治引领
2. 人才培养
3. 科技服务
4. 文化建设

### 八柱（八项举措）
- 政治引领：党委中心组学习、"三会一课"、主题党日 → "四级共学"体系
- 人才培养：本科生导师制、"五育并举"体系
- 科技服务：7支林业科技服务团、3个科技小院
- 文化建设：党建文化长廊、6个党员服务岗

## 实践成效

### 1. 党建引领强起来
- 教工第一党支部 → 第四批"全国党建工作样板支部"（已验收）
- 1个五星级支部、3个四星级支部
- 陈伏生教授获"全国优秀共产党员"称号
- 黄英老师获"全省模范教师"称号

### 2. 人才培育优起来
- 本科生升学率：10%（2012）→ 35%（近年）
- 一次性就业率：保持90%以上
- 西部计划录取人数：连续三年全校首位
- 国家级大创项目：近五年22项
- "挑战杯"省赛：特等奖1项、一等奖2项
- 中国国际大学生创新大赛：国赛银奖1项、铜奖1项

### 3. 科研服务强起来
- 国家重点研发计划项目：2项
- 梁希林业科学技术奖：二等奖3项
- SCI一区论文：27篇
- 获批蓝冠噪鹛国家保护研究中心
- 亚热带森林资源培育省级重点实验室

### 4. 社会影响大起来
- 7支林业科技服务团、3个科技小院
- 服务覆盖全省11个设区市
- 推广油茶、竹类等科技成果31项
- 服务面积超300万亩
- 培训林农2万余人次
- 完成秀美乡村规划项目15个

## 品牌体会

1. **党建和业务不能是两张皮** — 把党支部建在科室里，双带头人真正挑担子
2. **农科院校优势在田间地头** — 科研成果得落到山头田间才有真正价值
3. **品牌需要有人讲、有人传** — 身边故事最有说服力
