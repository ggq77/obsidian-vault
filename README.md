# Obsidian 知识库使用指南

## 目录结构

```
Obsidian知识库/
├── 0️⃣每日记录/      ← 每日工作日志、复盘
├── 1️⃣学生案例/      ← 典型案例积累（按类别归档）
├── 2️⃣政策文件/      ← 学习笔记、心得
├── 3️⃣工作流程/      ← SOP、模板
├── 4️⃣知识卡片/      ← 闪念笔记、碎片知识
├── 5️⃣阅读笔记/      ← 外部学习摘录
└── 6️⃣党建工作/      ← 党建品牌、工作总结、计划
```

## 双向链接使用

在笔记中使用 `[[笔记名称]]` 创建链接：

```
这是一个关于处分流程的笔记 [[处分流程]]

引用另一个笔记的某个标题：
[[学生案例#心理危机干预]]
```

## 标签使用

在笔记末尾添加 `#标签名`：

```
#心理 #危机干预 #工作流程
```

## Obsidian 下载地址

官网：https://obsidian.md

建议下载桌面版（Windows/Mac），数据存在本地。

- 提炼知识点
- 补充完善笔记
- 发现知识盲点
