# 开除学籍处分

## 基本信息

| 项目 | 内容 |
|------|------|
| 处分期限 | 永久（开除学籍） |
| 处分依据 | 《江西农业大学学生违纪处分办法》（赣农大发〔2024〕38号） |
| 学院权限 | 提出意见 |
| 学工处权限 | 审核 |
| 分管校领导 | 审核 |
| 最高权限 | 校长办公会决定 |
| 备案 | 报省教育厅备案 |

## 处分执行

1. 学生应在学校送达处分决定书后 **10个工作日内** 办理完相关手续离校
2. 档案退回其家庭所在地
3. 户口迁回原户籍地或家庭户籍所在地
4. 学校发给学习证明
5. 逾期不办理者，学校注销其在校内各种关系

## ⚠️ 与留校察看的关键区别

1. **学生监护人须在检讨书上签字**
2. 必须提供公安机关等部门的违法犯罪材料
3. 经分管校领导批准后，报**校长办公会**研究决定
4. 需报**省教育厅备案**

## 权限划分

| 步骤 | 主体 | 工作内容 |
|------|------|---------|
| 1 | 学院 | 调查取证 |
| 2 | 学院 | 教育告知 |
| 3 | 学院 | 党政联席会议，提出处理意见 |
| 4 | 学院 | 填写《学生处分登记表》，连同证据材料报送学工处 |
| 5 | 学工处 | 处务会进行合法性审查 |
| 6 | 分管校领导 | 审核 |
| 7 | 校长办公会 | 研究决定 |
| 8 | 学工处 | 形成《处分决定书》，报省教育厅备案 |
| 9 | 学院 | 处分送达（10个工作日内离校） |

## 报送材料要求

1. 检讨书（**须学生监护人签字**）
2. 《关于XXX同学违纪情况的说明》
3. 《学生处分登记表》
4. **违纪证据材料（如公安机关等提供的违法犯罪材料）**
