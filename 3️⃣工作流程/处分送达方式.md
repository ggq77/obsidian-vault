# 处分决定书送达方式

## 送达方式优先级

```
直接送达 > 邮寄送达 > 公告送达
```

## 1. 直接送达（首选）

- 将《处分决定书》直接送达学生本人
- 学生签字确认
- **最佳方式，优先采用**

## 2. 留置送达（学生拒绝签收）

- 学生拒绝签收时采用
- 须有**2名工作人员**在处分决定书上签字说明
- 学院记录在案

## 3. 邮寄送达

- 适用于已离校的学生
- 保留邮寄凭证

## 4. 公告送达（最后手段）

- 适用于难于联系的学生
- 利用学校网站、新闻媒体等公告
- **公告期为30天**
- 公告期满视为送达
