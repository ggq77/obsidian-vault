# 公文格式标准（GB/T 9704-2012）

## 页面设置

| 项目 | 规格 |
|------|------|
| 用纸 | A4（210mm × 297mm） |
| 上白边 | 37mm |
| 下白边 | 35mm |
| 左白边 | 28mm |
| 右白边 | 26mm |
| 版心尺寸 | 156mm × 225mm |
| 每页行数 | 22行 |
| 每行字数 | 28字 |

## 字体字号

### 版头
| 项目 | 字体字号 |
|------|---------|
| 份号 | 3号黑体 |
| 密级和保密期限 | 3号黑体 |
| 紧急程度 | 3号黑体 |
| 发文机关标志 | 小标宋体，字号酌定 |
| 发文字号 | 3号仿宋体 |
| 签发人 | 3号仿宋体 |

### 主体
| 项目 | 字体字号 |
|------|---------|
| 标题 | 2号小标宋体 |
| 主送机关 | 3号仿宋体 |
| 正文 | 3号仿宋体 |
| 附件说明 | 3号仿宋体 |
| 发文机关署名 | 3号仿宋体 |
| 成文日期 | 3号仿宋体 |
| 抄送机关 | 4号仿宋体 |
| 印发机关和印发日期 | 4号仿宋体 |

### 页码
- 4号半角宋体阿拉伯数字
- 左右各放一条一字线
- 一字线距版心下边缘7mm
- 格式：`- 1 -`

## 段落格式

### 标题
- 居中排布
- 上距版心35mm
- 下空一行编排主送机关

### 主送机关
- 顶格编排
- 最后一个名称后标全角冒号

### 正文
- 每自然段**左空二字**
- 回行顶格
- 数字、年份**不能回行**

### 附件说明
- 在正文下空一行
- 左空二字编排"附件"二字
- 后标全角冒号和附件名称

### 成文日期
- 右空四字编排
- 阿拉伯数字，年月日标全
- 月、日不编虚位（1不编为01）

## 行距
- **固定值28磅**
- 段前0磅，段后0磅

## 林学院学工办特定要求

### 发文字号格式
```
林学工发〔2026〕×号
```

### 套红文件
- 需要套红：违纪情况说明、处分决定书
- 不需要套红：检讨书（学生手写）
