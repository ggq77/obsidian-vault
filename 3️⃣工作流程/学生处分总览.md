# 学生违纪处分流程

> 依据：《江西农业大学学生违纪处分办法》（赣农大发〔2024〕38号）

## 四级处分对比

| 处分等级 | 处分期限 | 学院权限 | 是否报学工处 | 学位影响 |
|---------|---------|---------|------------|---------|
| [[警告或严重警告]] | 6个月 | 学院直接决定 | 仅备案 | 无 |
| [[记过]] | 12个月 | 学院提出意见 | 学工处决定 | 两次记过可不授学位 |
| [[留校察看]] | 12个月 | 学院提出意见 | 分管校领导决定 | 不授学位 |
| [[开除学籍]] | 永久 | 学院提出意见 | 校长办公会决定 | 离校 |

## 通用流程（6步）

```
调查取证 → 教育告知 → 党政联席会 → 报送/决定 → 处分送达 → 学工处备案
```

### Step 1 调查取证
学生违纪后的调查取证。

### Step 2 教育告知
书面告知学生事实、理由及依据，听取陈述和申辩。
形成材料：
- [[检讨书]]（学生提供）
- [[关于XXX同学违纪情况的说明]]（学院提供）

### Step 3 党政联席会（学院）
学院召开党政联席会议，提出处理意见/决定。

### Step 4 报送/决定
根据处分等级，决定程序：
- 警告/严重警告 → 学院直接决定
- 记过 → 学院报学工处决定
- 留校察看 → 学院→学工处→分管校领导
- 开除学籍 → 学院→学工处→校长办公会

填写：[[学生处分登记表]] + [[处分决定书]]

### Step 5 处分送达
- 直接送达学生本人签收
- 拒绝签收 → 留置送达（2名工作人员签字）
- 已离校 → 邮寄送达
- 难于联系 → 公告送达（30天）

### Step 6 学工处备案
报送材料：
1. 检讨书
2. 关于XXX同学违纪情况的说明
3. 学生处分登记表
4. 处分决定书
5. 违纪证据材料（留校察看及以上须提供）

## 申诉期限
学生可在收到《处分决定书》之日起 **10日内** 向学工处提出书面申诉。

## 档案归整
处分决定书、相关查证资料及解除处分材料，应真实完整地归入学生本人档案。
