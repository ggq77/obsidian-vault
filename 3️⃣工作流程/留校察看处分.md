# 留校察看处分

## 基本信息

| 项目 | 内容 |
|------|------|
| 处分期限 | 12个月（毕业班可缩短，不低于6个月） |
| 处分依据 | 《江西农业大学学生违纪处分办法》（赣农大发〔2024〕38号） |
| 学院权限 | 学院提出意见 |
| 学工处权限 | 审核 |
| 分管校领导 | 决定 |
| 学位影响 | 不授学位（不论是否解除） |

## 管理要求

- 处分期内不得担任学生干部（已担任的应予免职）
- 取消各类评优评先参评资格和奖励资格
- 停止享受各类奖励及相关荣誉
- 党、团员学生根据党纪、团纪给予相应处分

## 限授学位情形

依据《江西农业大学授予普通本科毕业生学士学位工作实施细则》（赣农大发〔2023〕18号）第三条第二款：

> 在校期间因课程考核违纪及其他原因受到"留校察看"处分者（**不论解除与否**），不授予学士学位。

## ⚠️ 与记过的关键区别

1. **必须提交违纪证据材料**（如《考场情况登记表》复印件等）
2. 需经分管校领导决定，不只是学工处
3. 学位影响更严重（直接不授学位，不论解除）

## 权限划分

| 步骤 | 主体 | 工作内容 |
|------|------|---------|
| 1 | 学院 | 调查取证 |
| 2 | 学院 | 教育告知、形成检讨书+违纪情况说明 |
| 3 | 学院 | 党政联席会议，提出处理意见 |
| 4 | 学院 | 填写《学生处分登记表》，连同证据材料报送学工处 |
| 5 | 学工处 | 处务会研究，报分管校领导 |
| 6 | 分管校领导 | 提出处理决定 |
| 7 | 学工处 | 形成《处分决定书》 |
| 8 | 学院 | 处分送达 |
| 9 | 学工处 | 备案 |

## 报送学工处材料

1. 检讨书
2. 《关于XXX同学违纪情况的说明》
3. 《学生处分登记表》
4. **违纪证据材料**（如《考场情况登记表》复印件等）
