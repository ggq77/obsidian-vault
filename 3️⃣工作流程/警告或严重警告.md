# 警告或严重警告处分

## 基本信息

| 项目 | 内容 |
|------|------|
| 处分期限 | 6个月 |
| 处分依据 | 《江西农业大学学生违纪处分办法》（赣农大发〔2024〕38号） |
| 学院权限 | 学院直接决定 |
| 学工处备案 | 需要 |

## 管理要求

- 处分期内不得担任学生干部（已担任的应予免职）
- 取消各类评优评先参选资格
- 停止享受各类奖励及相关荣誉
- 党、团员学生根据党纪、团纪给予相应处分

## 权限划分

- **主管部门**：学生工作部（处）
- **决定权限**：由学生所在学院研究决定

## 与其他处分级别的区别

- 权限最小，学院可直接决定
- 处分期限最短（6个月）
- 不影响学位授予
- 证据材料非强制提交学工处
