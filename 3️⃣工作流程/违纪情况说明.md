# 关于XXX同学违纪情况的说明（学院用）

## 基本信息

| 项目 | 内容 |
|------|------|
| 制定方 | 学院 |
| 撰写时间 | 调查取证后、党政联席会前 |
| 用途 | 报送学工处备案/决策的重要材料 |

## 内容框架

```
关于XXX同学违纪情况的说明

一、基本情况
学生姓名、学号、班级、院系

二、违纪事实
时间、地点、经过、证据

三、错误性质认定
违反哪条校纪校规

四、学院处理意见
党政联席会讨论结果

五、附件清单
证据材料清单
```

## 注意事项

- 事实陈述要准确，有证据支撑
- 措辞客观公正，不夸大不缩小
- 学院的处理意见要明确
