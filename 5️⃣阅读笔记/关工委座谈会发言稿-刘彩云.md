# 关工委座谈会发言稿 — 刘彩云

> 场合：教育部关工委调研座谈会
> 发言人：刘彩云（江西农业大学林学院/园林与艺术学院）

## 基本情况介绍

- 二级学院关工委成立于2025年2月
- 全校第一个二级学院关工委
- 现有成员15人，"五老"同志6人（占四成）
- 机制："党委领导、老同志主导、师生参与"

## 核心人物：杜天真教授

| 信息 | 内容 |
|------|------|
| 身份 | 国家级教学名师、关工委常务副主任 |
| 年龄 | 86岁 |
| 教龄 | 60余年 |
| 贡献 | 2021年捐资110万元设立林业教育基金 |
| 坚守 | 退休后一直坚守在育人一线 |

**金句：** "把养老钱变成未来。"

**主编：**《三实农技》科普季刊，坚持21年，免费发放15万多份，农民称"致富宝典"

## 七大做法

### 1. 搭建"老少双向"平台
- 1个"五老"工作室常年运行
- 新进教师座谈会、退休教师新春座谈会
- 青年学生上门看望独居老同志
- 不是单向关怀，是双向奔赴

### 2. 发挥专业优势，银发力量扎根大地
- 老专家带学生进林区、下田间
- 把课堂搬到山上地里
- 让学生在实践中了解林业、热爱林业

### 3. 传承师德精神
- 杜老师批改论文从标点符号改起
- 金句："一个人的治学态度，比学术能力更重要"
- 培养的三位女博士（郭晓敏、刘苑秋、张露）都成为省教育系统先进典型
- 名言："教育，就是一棵树摇动另一棵树，一朵云推动另一朵云"

### 4. 关爱学生成长
- 施兴华教授缅怀会（2024年3月）
- 施老师一生未婚，把全部心血奉献给学生
- 患病期间年轻教师自发轮流照顾
- "这是学生上过最生动的一堂课"

### 5. 设立学生委员
- 《林学院/园林与艺术学院关工委学生委员工作条例》
- 学生委员由分团委副书记（学生）或学生会主席担任
- 职责：思想调研、学风建设、心理健康关注、社会实践组织
- 桥梁连接"五老"与青年学生

### 6. 深入开展"读懂中国"活动
- 挖掘杜天真教授典型事迹
- 微视频、征文作品
- **荣获教育部关工委"读懂中国"活动最佳微视频、最佳征文**（全省高校唯一同时获得两项最高奖项）
- 身边人讲好身边故事，身边事教育身边人

### 7. 创新支部共建机制
- 退休四支部与学生党支部结对共建
- 设立老教授工作室
- "老带青、师带徒"传帮带机制
- 学术座谈、科研指导、职业规划辅导

## 2026年工作打算

1. **强化政治引领** — 纳入学院党建整体布局
2. **夯实组织基础** — 建立"五老"资源库
3. **打造特色品牌** — "林学银发专家进课堂"品牌活动
